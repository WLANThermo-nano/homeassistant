# Troubleshooting

Here you will find help for problems with the integration.

## Troubleshooting
- Entities do not appear: Check IP, restart integration
- Notification issues: Check token and user key
- Bluetooth sensors not found: Restart integration
- Get help in the [WLANThermo Forum](https://wlanthermo.de/forums/) or open a GitHub issue
