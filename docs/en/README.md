## Overview
This integration connects Home Assistant with WLANThermo devices and provides real-time sensor data, system diagnostics, and advanced control features. Supported models are Nano V1+, V3, Mini V2 ESP32, V3, and Link V1.

## Features
- Automatic detection and setup
- Dynamic temperature and pitmaster sensors
- Remaining time calculation for each channel
- Notification configuration (Telegram, Pushover)
- Support for Bluetooth temperature sensors
- Cloud diagnostics
- System diagnostics (WiFi, battery, charging)
- Channel color configuration
- Full translation support (DE/EN)
- Configurable scan interval
- Offline-tolerant startup
- Options flow for advanced settings
